This website started off as a carrd, prob in 2021, that doesnt exist anymore. too limiting. We then moved to neocities, but it has a file limit. Github repositories don't have limits so're here since july 2022.
Bc we're indecisive and out tastes change with time, this place's been remade 8 times if i'm not mistaken. I want(ed) this site to be simple, but still interesting and different. Very soon i realized i had to put stuff in this website, so i put texts and drawings, but this place itself became a reason for it to exist, yk? to experiment with different colors and fonts, making each room different. Because it's nowhere in particular. It also feels p open with few borders.
